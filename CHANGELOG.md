# Changelog

## 0.1.0

Initial public project layout.

- Sky-compatible Windows Window2 helper protocol.
- Standalone stdio, framed stdio, and native-pipe transports.
- Window/app discovery, WGC screenshots, UIA accessibility trees, and input
  actions.
- Approval, request-budget, and turn-interruption handling.
- Rust unit tests, WinForms fixture runners, and optional original-vs-Rust A/B
  runners.
